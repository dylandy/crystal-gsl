# statistics

GNU Scientific Library (GSL) binding for Crystal

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  gsl:
    github: dylandy/crystal-gsl
```
## System dependencies

### Ubuntu < 16.04 

* libgsl0-dev 
* gsl-bin

### Ubuntu >= 16.04

* libgsl-dev
* gsl-bin

## Usage


```crystal
require "gsl"
```

Warning:

- Not fully test
- Pre-release (API will break)
- Not fit for production
